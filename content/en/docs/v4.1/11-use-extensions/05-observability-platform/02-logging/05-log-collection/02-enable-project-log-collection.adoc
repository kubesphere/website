本节介绍如何在项目中启用日志收集。
//note
[.admon.note,cols="a"]
|===
|说明

|
启用后，在项目中创建工作负载和任务时，**存储设置**页签的**收集卷上日志**功能将可用。
|===


== Prerequisites


* {empty}
include::../../../../../_custom-en/projectManagement/projectManagement-prer-requiredPermission_new.adoc[]



* {ks_product_right}平台需要安装并启用 **WhizardTelemetry 日志**扩展组件，并已link:../01-enable-logsidecar/[启用日志收集]。

== Steps

include::../../../../../_custom-en/projectManagement/projectManagement-oper-openProjectPage.adoc[]

+

include::../../../../../../_ks_components-en/oper-navigate.adoc[]

+

. 在**收集卷上日志**区域，点击切换开关将功能状态设置为**已启用**。
