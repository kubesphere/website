:ks_menu: **概览**


本节介绍如何查看集群的概览信息。


== 前提条件

:relfileprefix: ../../../

include::../../../_custom-en/clusterManagement/clusterManagement-prer-requiredPermission_new.adoc[]

:relfileprefix: ./

== 操作步骤

. 以具有pass:a,q[{ks_permission}]权限的用户登录{ks_product-en} Web 控制台并进入您的集群。
. 在左侧导航栏选择pass:a,q[{ks_menu}]。

+
--
[%header,cols="1a,4a"]
|===
|功能区 |描述

|名称、标签和描述
|
显示当前集群的名称、标签和描述。

* 成员集群的名称由用户添加成员集群时设置。

* 集群名称右侧显示当前集群的标签。

|基本信息
|显示当前集群的提供商、Kubernetes 版本、{ks_product-en}版本和对企业空间的可见性。

|已创建资源数量
|显示当前集群的资源数量，如项目、容器组、部署、有状态副本集、守护进程集、任务、定时任务、持久卷、服务、应用路由、成员、角色等的数量。

|工具
|显示 kubectl 命令行工具，和 kubeconfig 配置文件。

|===
