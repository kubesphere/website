本节介绍编辑集群信息。

您可以编辑集群的标识、提供商和描述。{ks_product-en}不支持编辑已添加集群的名称。


== 前提条件

您需要在{ks_product-en}平台具有pass:a,q[{ks_permission}]权限。



== 操作步骤

include::../../../../_custom-en/platformManagement/platformManagement-oper-logIn.adoc[]

. 点击**集群管理**。

. 在需要操作的集群右侧点击image:/images/ks-qkcp/zh/icons/more.svg[more,18,18]，然后在下拉列表中选择**编辑信息**。

. 在**编辑信息**对话框，设置集群的标识、提供商和描述，然后点击**确定**。
+
--
[%header,cols="1a,4a"]
|===
|参数 |描述

|标识
|集群的用途标识。

|提供商
|集群的提供商。在下拉列表中选择提供商。

|描述
|集群的描述信息。描述可包含任意字符，最长 256 个字符。
|===
