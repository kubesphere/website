--
[%header,cols="1a,4a"]
|===
|参数 |描述

|集群
|定制资源定义所属的集群。

. 在定制资源定义详情页面右侧的**定制资源定义**页签查看使用定制资源定义创建的定制资源。

--
[%header,cols="1a,4a"]
|===
|参数 |描述

|名称
|定制资源的名称。

|项目/DevOps 项目
|定制资源所属的项目或 DevOps 项目。

|创建时间
|定制资源的创建时间。
