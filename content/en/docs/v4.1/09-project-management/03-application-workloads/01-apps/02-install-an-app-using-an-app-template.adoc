本节介绍如何从企业空间或第三方应用仓库中的应用模板安装应用。

//note
[.admon.note,cols="a"]
|===
|说明

|
若要使用企业空间的应用模板，需要安装并启用“应用商店”扩展组件。安装后，企业空间的**应用管理**菜单下将显示**应用模板**选项。然后由具有相关权限的企业空间成员创建企业空间的应用模板。

|===

. 在**应用模板**页面，点击**当前企业空间**下拉列表，选择**当前企业空间**或一个应用仓库。

. 点击需要使用的应用模板，然后点击**安装**。
