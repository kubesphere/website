本节介绍如何设置飞书通知。

飞书通知功能支持通过以下两种方式向用户发送通知：

* **会话设置**：使用飞书应用向特定用户或部门发送通知。

* **群机器人设置**：使用飞书机器人向特定群聊发送通知。

* 您需要提前从飞书平台获取应用或群机器人的访问参数。有关更多信息，请参阅link:https://open.feishu.cn/document/ukTMukTMukTM/uITNz4iM1MjLyUzM[飞书开发者文档]。
