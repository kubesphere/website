本节介绍如何回退微服务配置。

每次编辑微服务配置都会生成修改记录，您可以使用修改记录将微服务配置回退到历史版本。


== 前提条件

* {ks_product_right}平台需要安装并启用 **KubeSphere Spring Cloud** 扩展组件。
