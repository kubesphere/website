本节介绍如何编辑微服务实例权重。

微服务实例的权重用于决定微服务实例的流量分配，发送给微服务的流量将根据微服务实例的权重分配给微服务实例。


== 前提条件

* {ks_product_right}平台需要安装并启用 **KubeSphere Spring Cloud** 扩展组件。
