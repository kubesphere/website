KubeSphere 的项目即 Kubernetes 的命名空间。项目有两种类型，即单集群项目和多集群项目。单集群项目是 Kubernetes 常规命名空间，多集群项目是跨多个集群的联邦命名空间。项目管理员负责创建项目、设置限制范围、配置网络隔离以及其他操作。
