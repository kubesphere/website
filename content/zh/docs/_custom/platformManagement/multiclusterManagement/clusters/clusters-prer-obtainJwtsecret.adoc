. 在 **kubectl** 对话框，执行以下命令获取主集群的 jwtSecret 值。
+
--
// Bash
include::../../../../../_ks_components/code/bash.adoc[]

kubectl -n kubesphere-system get cm kubesphere-config -o yaml | grep -v "apiVersion" | grep jwtSecret
