在 KubeSphere 中，您可以使用交互式 Web 控制台或内置的原生命令行工具 `kubectl` 来设置集群并配置其功能。作为集群管理员，您将负责一系列任务，包括在节点上管理调度并添加标签、控制集群可见性、​​监控集群状态、设置集群的告警规则和通知规则，以及配置存储和日志收集解决方案等。

{{< notice note >}}

本章未介绍多集群管理。有关此功能的更多信息，请参见[多集群管理](../multicluster-management/)。

{{</ notice >}}
